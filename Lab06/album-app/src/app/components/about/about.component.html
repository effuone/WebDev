About Album App

Angular Route and HTTP Module Implementation

This application demonstrates the use of Angular's Route and HTTP modules.

The app retrieves album and photo data from JSONPlaceholder, a free fake online REST API for testing and prototyping.

Features:
  • Viewing a list of albums
  • Viewing album details
  • Creating new albums
  • Updating album titles
  • Deleting albums
  • Viewing photos for each album
